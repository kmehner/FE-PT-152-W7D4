Introduction to CSS

Why use CSS?

CSS is a styling language that allows us to style HTML elements to make web pages more visually appealing.

Example: "Hey, I want my paragraphs to be blue and centered."

Three Ways to Add CSS

  1. Inline CSS: This text is blue (inline style).
  2. Internal CSS: Styled in the style tag within the head section.
  3. External CSS: Linked through an external stylesheet.

CSS Syntax

Selectors target HTML elements, and properties define how those elements will look.

selector {
    property: value; 
    property: value; 
}

h1 {
    color: blue; 
}
